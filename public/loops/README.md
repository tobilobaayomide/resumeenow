Place your recorded step-loop videos in this folder using these exact filenames:

- `step-01.mp4`
- `step-02.mp4`
- `step-03.mp4`

Recommended settings:

- 5-8 seconds
- 1920px width
- 24fps
- no audio
- H.264 `.mp4` (`yuv420p`)
- target visually clean UI capture quality; avoid aggressive compression

If a file is missing, landing sections automatically fall back to the demo placeholder video.
